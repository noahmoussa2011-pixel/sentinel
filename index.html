System monitoring
that stays out of your way.

Sentinel is a lightweight Linux system monitor. Live graphs, per-core CPU tracking, a full process manager, disk and network I/O — all in a clean GTK4 interface. No Electron. No web stack. No nonsense.

Installation

Up and running in 60 seconds.

Works on any Debian or Ubuntu based distro. ARM64 and amd64.

01
Download the package
wget https://github.com/noahmoussa2011-pixel/sentinel/releases/download/v1.0.0/sentinel_1.0.0_all.deb click to copy
02
Install dependencies
sudo apt install -y python3-gi python3-gi-cairo gir1.2-gtk-4.0 python3-psutil click to copy
03
Install Sentinel
sudo dpkg -i sentinel_1.0.0_all.deb click to copy
04
Launch
sentinel click to copy
Features

Everything you actually need.

▤
Live Dashboard
CPU, RAM, swap, disk I/O, network and uptime — all on one screen, updating every second.
◫
Per-Core CPU
Individual usage bar for every core. Scrolling history graph with configurable time window.
◈
Memory Breakdown
Used, cached, buffers and free — shown as live bars. Swap tracking included.
⬡
Process Manager
Full process list sorted by CPU. Search by name or PID. Select a row and hit End Task to kill it.
↑↓
Network Monitor
Upload and download speed with live graph. Interface info, IPs, and session totals.
⊟
Disk I/O
Read and write speeds per second. Partition usage bars for every mounted filesystem.
